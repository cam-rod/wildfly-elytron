WildFly Elytron at Open Source Day 2022!

Welcome Open Source Day participants! We’re glad to have you join us for the hackathon.

To get started, follow the steps below:

  1. Setup the necessary tools to get started. We use Java 11, Maven, and GitHub for development. Follow the setup instructions for your operating system:

    • Linux & WSL
    • Windows 10/11
    • macOS

  2. Pick your IDE. If it’s your first time, we recommend an IDE which includes all the tools you’ll need, like Intellij IDEA or Eclipse. For those more familiar with Java development and using Maven, feel free to use whatever you’d like.

  3. Get familiar with WildFly. Our Getting Started page has some resources to help you learn more about the project. You don’t need to go through all of the links, so here are a few sections to look at:

    • WildFly Project Introductions

    • Getting familiar with Elytron

    • Building and testsuite comamnds

  4. Start contributing! Head over to GitHub to get started. The file CONTRIBUTING.md has everything you need to get started, including how to use our issue tracker (link), open a pull request, and get your changes into Elytron!

If you have any questions, send us a message on Slack.
